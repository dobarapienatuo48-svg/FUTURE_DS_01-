## 🎯 Objectif du Projet

Analyser des données de ventes commerciales pour identifier :
- Les **tendances de revenus** sur l'année 2024
- Les **produits les plus performants** et leur contribution au revenu
- La **performance par région** et opportunités de croissance
- Le **classement des vendeurs** et meilleures pratiques
- Des **recommandations actionnables** basées sur les données

## 💡 Insights Principaux

1. **Produit Star** : L'Ordinateur Portable génère le revenu le plus élevé → augmenter les stocks
2. **Région Leader** : Abidjan domine → déployer ses stratégies dans les autres régions
3. **Marge globale** : ~35% → surveiller les remises accordées
4. **Saisonnalité** : identifier les pics pour mieux planifier les stocks
